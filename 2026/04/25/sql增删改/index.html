sql增删改

参考https://www.runoob.com/mysql/mysql-create-tables.html 以及b站橙子科技视频

mysql命名规则

  • 必须是字母或者下划线_开头
  • 不能是纯数字(一定要用纯数字需要加反引号**`**)
    • **`**用于包裹字符串值
  • 后续字符可以是字母,数字,下划线,美元符号

数据库

  • select database(); #确认所在库
  • mysql -u root -p #输入密码登录数据库
  • show databases; #查看数据库
  • create database databases_name charset utf8; #创建数据库databases_name并选择字符集
  • drop database database_name; #删除数据库
  • use database_name; #选择进入数据库database
  • create table table_name ( column1 datatype, column2 datatype, ...) #创建数据表database(如果数据表内没有列数据将无法创建成功)
    • 相关参数以及格式:
    • 除了数字类型不用加单引号,其余类型插入需要加单引号
    • int ,bigint :存放整数,直接写数字,**不加引号
    • decimal(m,d) :精确小数,m代表最多存几位数字,d代表小数有几位.**不加引号
    • float :近似小数,存储不精确,适合科学计算.
    • varchar(n) :可变长字符,最多存n个字符,省空间
    • char(n) :定长字符串,总占n个字符空间,不足补空格,查询时会去掉尾部空格
    • text :长文本,不限长度
    • date :只存年-月-日 ->‘YYYY-MM-DD’
    • datetime :存年-月-日 时:分;秒 ->‘YYYY-MM-DD HH:MM:SS’
    • timestamp :类似与datetime ,但会受时区影响,常用来记录自动生成时间 ->‘YYYY-MM-DD HH:MM:SS’
  • show full columns from table_name; #查看数据表的列数据
  • select *from table_name; #查看数据表的行数据
  • drop table table_name; #删除数据表
  • rename table table_name to new_table_name; #修改数据表名称为新名字
  • alter table user character set utf8; #修改字符级

数据列和数据行

  • `insert into table_name
    (
    列数据1,列数据2…)
    values
    (行数据1,行数据2…)
    ;
  • `alter table table_name add 列名称 decimal(m,d); #增加一列内容,decimal是小数类型
  • update table_name set 列名称=5000; #修改改列的所有数据变为5000
  • update table_name set name='benben',列名称='...' where id=1; #限定修改,将id=1的行name修改为benben以及另一个行的数据修改成…
  • alter table table_name drop 列名称; #删除某一列
  • delete from table_name where 列名称='...' #删除列名称='…'的那一行
  • delete from table_name #删除整张数据表

数据库查询

  • select * from users where id=1; #select+列名(*代表所有)from+表名 where+条件语句
  • select * from users where id in('3'); #从users表格里,查询所有包含id=3
  • select *from users where id=(select id from users wher username=('admin')); #子查询 ,优先执行()的语句, 查找在users表里面叫admin的行的id,查找这个id 的行信息

union语句

  • select id from users union select email_id from emails; #查询并合并数据显示
  • select * from uses where id=6 union select * from emails where id=6; #(该例子中第一个数据表行数有3行,后一个数据表行数只有两行)uinon语句要求前后查询的数据表行数必须一致,否则会报错
  • 改进方案:
    ``select * from uses where id=6 union select ,3 from emails where id=6;` #可以在后面加个数字,这个数字会成为填充行,将3填入与前一个表格没有对应的行中(不止可以填3,其他数据类型也可以,但是记得加单引号)
