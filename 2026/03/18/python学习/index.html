数据类型

类型 描述 说明
string 字符串类型 用引号引起来的数据都是字符串
int 整型 数字类型,存放整数
float 浮点型 数字类型,存放小数

float(x) 将对象x转换为一个浮点数
str(x) 将对象x转换为一个字符串

标识符

  • 标识符:用户在编程时用于给变量,类,方法等命名

  • 标识符命名规则

    • 只允许出现英文,中文,数字,下划线(_)其余任何内容都不被允许
      (不推荐中文,数字不可以开头)
    • 大小写敏感(有大小写区分)
    • 不可使用关键字

  • 变量命名规范

    • 见名知意
    • 多个单词组合变量名,用下划线分割
    • 英文字母全小写

    运算符

  • 数学运算符

运算符 描述
= 加
- 减
* 乘
/ 除
// 取整数
% 取余
** 指数
  • 赋值运算符
    • =
  • 复合赋值运算符
运算符
+=
-=
*=
/=
%=
**=
//=
  • 逻辑运算符
    • and：x and y (有0就是0，没有0返回y值)
    • or： x or y（有一个不是0就不是0）
    • not： not x （如果x是0，返回1，反之返回0）
  • 成员运算符
    • in：如果在指定的序列中找到值返回 1，否则返回 0
    • not in:如果在指定的序列中没有找到值返回 1，否则返回 0
  • 身份运算符
    • |运算符|描述|实例|
      |—|—|—|
      |is|is 是判断两个标识符是不是引用自一个对象|x is y, 类似 id(x) == id(y) , 如果引用的是同一个对象则返回 True，否则返回 False|
      |is not|is not 是判断两个标识符是不是引用自不同对象|x is not y ， 类似 id(a) != id(b)。如果引用的不是同一个对象则返回结果 True，否则返回 False。|

字符串

  • 字符串的三种定义方式

    • 单引号定义法
    • 双引号定义法
    • 三引号定义法(和多行注释的写法一致,区别是是否用变量接收,且支持换行操作)

  • 字符串的引号嵌套

    • 单引号定义法
    • 双引号定义法
    • 用转义字符()将引号解除效用,变成普通字符串
      • 举例: “"注意转义字符位置"”
        • 输出: “注意转义字符位置”

  • 字符串的拼接

    • 字面量和变量或者变量和变量之间通过"+"拼接完成
    • 拼接只能适用于字符串

  • 字符串索引

    • str = ‘Hello World!’ print str # 输出完整字符串

    • print str[0] # 输出字符串中的第一个字符

    • print str[2:5] # 输出字符串中第三个至第六个之间的字符串

    • print str[2:] # 输出从第三个字符开始的字符串

    • print str * 2 # 输出字符串两次

    • print str + “TEST” # 输出连接的字符串

  • 字符串格式化

    • 语法

      • **%**表示: 我要占位
      |格式符号|转化|
|---|---|
|%s|将内容转换成字符串,放入占位位置|
|%d|将内容转换成整数,放入占位位置|
|%f|将内容转换成字浮点型,放入占位位置|

    • 写法

      • 在字符串之后空格写**%,然后在需要拼接字符串的地方写上%s**,如果有多个拼接的变量,需要用**()**将变量括起来

    • 数字精度控制

      • 我们可以使用**“m.n”**来控制数据的宽度和精度
      • m: 控制宽度,设置宽度小于数字本身时不生效(小数点也算一点宽度)
      • .n: 控制小数点精度,要求是数字,会对小数四舍五入

  • 快速格式化

    • 通过语法: f"内容{变量}"的格式来快速格式化
    • 特点: 不限数据类型,不做精度控制
